# عيادات بيورا | Bura Clinic Website

موقع React + Vite + TypeScript بتصميم فاخر وواجهة عربية أولًا لعيادات بيورا في الرياض.

## ما تم ضبطه

- توحيد الهوية إلى: **عيادات بيورا / Bura Clinic**.
- تحديث بيانات التواصل في ملف مركزي: `src/config/clinic.ts`.
- استبدال الخدمات التجريبية بخدمات أقرب لنطاق بيورا: الجلدية، طب التجميل، الليزر، الحقن، جراحة التجميل، وجراحة اليوم الواحد.
- إزالة الأرقام الوهمية والبريد التجريبي من الواجهة.
- إصلاح زر “اعرفي المزيد” في كروت الخدمات ليفتح صفحة الخدمات.
- إصلاح مفاتيح الترجمة الناقصة في قسم تجربة المراجعين.
- تحويل معرض النتائج إلى نماذج عرض مؤقتة مع تنبيه واضح حتى تتم إضافة صور حالات حقيقية معتمدة.
- تحسين النصوص العربية والإنجليزية في الصفحات الداخلية.
- إزالة كلمة مرور لوحة التحكم الثابتة من الكود. لوحة التحكم لا تعمل إلا بعد إضافة متغير بيئة.

## التشغيل المحلي

```bash
npm install
npm run dev
```

## بناء نسخة الإنتاج

```bash
npm run build
```

## بيانات العيادة

كل بيانات التواصل الأساسية موجودة هنا:

```ts
src/config/clinic.ts
```

راجع خصوصًا:

- `phoneDisplay`: الرقم الظاهر للاتصال.
- `whatsappNumber`: رقم واتساب بصيغة دولية بدون +.
- `email`: البريد الإلكتروني.
- `addressAr` و `addressEn`: العنوان.
- `instagram`: رابط حساب إنستغرام.

## تفعيل لوحة التحكم اختياريًا

لوحة التحكم موقوفة افتراضيًا حتى لا توجد كلمة مرور مكشوفة داخل الكود. لتفعيلها أضف متغير البيئة التالي في الاستضافة:

```bash
VITE_ADMIN_PASSWORD=ضع_كلمة_مرور_قوية_هنا
```

ملاحظة: لوحة التحكم الحالية تحفظ التعديلات في المتصفح عبر `localStorage` فقط، لذلك لا تعتبر CMS دائمًا. للاستخدام الحقيقي اربطها بباكند أو CMS.

## ملاحظات قبل النشر النهائي

- تأكد من رقم الواتساب النهائي من العميل قبل إطلاق الحملات الإعلانية.
- أضف صور الأطباء الرسمية وصور قبل/بعد المعتمدة كتابيًا.
- راجع أي ادعاءات طبية أو عروض مع الفريق القانوني/التسويقي قبل النشر.
